Greenhouse Mental Health Project Documentation

This page provides a summary of the Greenhouse for Mental Health Development project and links to key documentation.

Project Summary

The goal of the Greenhouse for Mental Health project is to establish greenhousementalhealth.org as a premier digital ecosystem for mental health development. The platform aims to integrate diverse services and resources to empower both patients and professionals in a trusted, supportive, and accessible environment.

The project is being developed using a phased approach, starting with discovery and audit, moving through information architecture, UX and visual design, content strategy, and technology implementation. The technology stack is based on Wix, with Velo for custom logic, and includes integrations for scheduling, telehealth, and payments.

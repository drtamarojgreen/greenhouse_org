# Empirical MeSH Term Discovery Pipeline

This pipeline discovers dominant MeSH terms related to mental health over time using data-driven, unsupervised methods.

## Structure
- `config/`: Configuration parameters.
- `data/`: Raw, interim, and output data.
- `src/`: Source code for the pipeline.
- `scripts/`: Execution scripts.
- `logs/`: Runtime logs.

## Usage
```bash
python scripts/run_pipeline.py --config config/pipeline.yaml
```

## ðŸ“… Why do the records stop at 1989?

The data currently present in these results was generated using the **initial files of the 2025 PubMed Baseline** (`pubmed25n0001.xml.gz` and `pubmed25n0002.xml.gz`).

PubMed baseline files are distributed according to their PubMed Identifiers (PMIDs). Since PMIDs are assigned sequentially as articles are indexed, the earliest files in the baseline contain the oldest records.
- File **0001** contains the very first indexed articles, starting from the mid-19th century and 1940s/50s.
- File **0002** continues this sequence into the late **1980s**.

To analyze data up to 2024/2025, the pipeline would need to process subsequent files in the baseline (0003 through ~1300+). The current sample demonstrates the functional pipeline logic using the first two volumes of global medical research history.
